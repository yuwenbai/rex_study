该目录下的文件为OC静态文件以及对应的头文件，作为提供前端根据IPV4地址转换IPV6地址的功能。

使用流程：如果不需要进行V6网络适配，按照现有的NetWorkerImpl的使用逻辑即可。当需要进行V6网络适配的时候，调用TcpClienter.connectByV6
方法即可。内部实现逻辑是优先使用V4IP进行两次的尝试连接，在失败的时候通过调用OC静态文件进行IP转换并自动使用V6IP进行一次连接尝试，
如果失败将还原当前环境并且进入连接失败事件。

后期维护：在TcpClienter类中封装了单独的region模块 模块名为V6网络适配模块，当不需要相应内容的时候，直接将该模块注释即可。
